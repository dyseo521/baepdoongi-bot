# IGRUS 회비 안내

## 회비 금액

### 정규 회원
- **학기당 회비: 20,000원**
- 납부 시기: 매 학기 초 (3월, 9월)
- 납부 기한: 학기 시작 후 2주 이내

### 신입 부원
- 첫 학기: 20,000원 (정규 회비와 동일)
- 입회비: 없음

## 회비 납부 방법

### 계좌이체
- 은행: 카카오뱅크
- 계좌번호: 3333-00-0000000
- 예금주: 황도연
- 입금자명: 학번이름 (예:20홍길동)

### 확인
- 납부 후 Slack #회비 채널에 인증 메시지 남기기
- 재무팀에서 확인 후 회원 명단 업데이트

## 회비 사용 내역

### 주요 지출
- 스터디/세미나 간식비
- 동아리 행사 (MT, 네트워킹 등)
- 서버 운영비
- 비품 구입

### 투명한 운영
- 매 학기 말 회비 사용 내역 공개
- Slack #공지 채널에서 확인 가능
