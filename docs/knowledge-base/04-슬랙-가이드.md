# IGRUS Slack 사용 가이드

## 프로필 설정

### 표시 이름 규칙
반드시 아래 형식으로 설정해주세요:

```
이름/학과/학번
```

예시:
- 김아그/컴퓨터공학과/26
- 최인하/전기전자공학부/24

## 뱁둥이 봇 사용법

### 질문하기
@뱁둥이를 멘션하면 동아리 관련 질문에 답변해드려요!

예시:
- "@뱁둥이 회비가 얼마야?"
- "@뱁둥이 동아리방 어디야?"
- "@뱁둥이 다음 세미나 언제야?"

### 슬래시 커맨드
- `/가이드` - 동아리 활동 가이드 보기
- `/익명건의` - 익명으로 건의사항 제출

## 에티켓

### 권장 사항
- 질문 전 검색 먼저
- 답변에 감사 인사
- 건설적인 피드백
- 정보 공유 적극 참여

### 금지 사항
- 광고/스팸
- 정치/종교 논쟁
- 불쾌감을 주는 내용
