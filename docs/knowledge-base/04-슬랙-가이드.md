# IGRUS Slack 사용 가이드

## 프로필 설정

### 표시 이름 규칙
반드시 아래 형식으로 설정해주세요:

```
이름/학과/학번
```

예시:
- 홍길동/컴퓨터공학과/20
- 김영희/정보통신공학과/21


## 주요 채널

### 공지 채널
- **#공지** - 동아리 공식 공지 (읽기 전용)
- **#회비** - 회비 납부 인증

### 소통 채널
- **#general** - 일반적인 대화
- **#random** - 자유로운 주제
- **#질문** - 기술 질문, 도움 요청

### 활동 채널
- **#스터디** - 스터디 모집/공유
- **#프로젝트** - 프로젝트 논의
- **#취업** - 취업/인턴 정보

### 관심사 채널
- **#frontend** - 프론트엔드
- **#backend** - 백엔드
- **#mobile** - 모바일 개발
- **#ai-ml** - AI/머신러닝
- **#알고리즘** - 알고리즘/코테

## 뱁둥이 봇 사용법

### 질문하기
@뱁둥이를 멘션하면 동아리 관련 질문에 답변해드려요!

예시:
- "@뱁둥이 회비가 얼마야?"
- "@뱁둥이 동아리방 어디야?"
- "@뱁둥이 다음 세미나 언제야?"

### 슬래시 커맨드
- `/가이드` - 동아리 활동 가이드 보기
- `/익명건의` - 익명으로 건의사항 제출

## 에티켓

### 권장 사항
- 질문 전 검색 먼저
- 답변에 감사 인사
- 건설적인 피드백
- 정보 공유 적극 참여

### 금지 사항
- 광고/스팸
- 정치/종교 논쟁
- 불쾌감을 주는 내용
