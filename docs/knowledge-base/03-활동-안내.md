# IGRUS 활동 안내

## 정기 활동

### 정기 총회
- 일시: 매 학기 초/말
- 내용: 학기 계획 발표, 회고, 임원 선출

### 현재 진행 중인 스터디
- 알고리즘 스터디 (코딩테스트 대비)
- React 스터디 (프론트엔드)
- Spring Boot 스터디 (백엔드)
- ML/AI 스터디 (머신러닝)

### 스터디 참여 방법
1. Slack #스터디 채널에서 모집 공고 확인
2. 해당 스터디 리더에게 DM
3. 첫 모임 참석

## 프로젝트

### 동아리 프로젝트
- 매 학기 팀 프로젝트 진행
- 학기 말 데모데이 발표
- 우수 프로젝트 시상

### 외부 대회
- 해커톤 참가
- 공모전 참가
- 팀 빌딩 지원

## 행사

### MT (Membership Training)
- 시기: 학기 초 (보통 3월, 9월)
- 장소: 근교 펜션 또는 MT 장소
- 내용: 친목 도모, 팀 빌딩, 레크리에이션

### 네트워킹
- 졸업생 멘토링
- 기업 연계 세미나
- 취업/인턴 정보 공유

### 송년회/종강 파티
- 시기: 학기 말
- 내용: 회고, 시상, 친목
